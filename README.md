# Skill Analytics System (ML Based)

## 📌 Description
This is a Machine Learning based web application built using Django that analyzes user skills and provides insights and recommendations.

## 🚀 Features
- Skill analysis using ML
- User-friendly dashboard
- Data visualization
- Recommendation system

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- Machine Learning (Scikit-learn, Pandas)
- MySQL

## 📂 How to Run
1. Clone the repository
2. Install requirements
3. Run server
